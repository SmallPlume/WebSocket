# WebSocket
主要使用SpringMVC、WebSocket等写个小的在线聊天demo
#配置 版本 ：
  - spring:4.1.7.RELEASE;
  - shiro:1.2.5;
  - mybatis:3.3.0;
  - druid:1.0.25;
  
#  项目URL：http://localhost:8080/WebSocket/index
